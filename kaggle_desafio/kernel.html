Definio do problema

Descrio obtida do link do datset: Adult Data Set

Resumo: Prever se a renda excede US$50 mil/ano com base nos dados do censo. Tambm conhecido como conjunto de dados Renda do Censo.

Informaes do conjunto de dados:

A extrao foi feita por Barry Becker do banco de dados do Censo de 1994. Um conjunto de registros razoavelmente limpos foi extrado usando as seguintes condies: ((AAGE> 16) && (AGI> 100) && (AFNLWGT> 1) && (HRSWK> 0))

A tarefa de previso determinar se uma pessoa faz mais de 50 mil a ano.
